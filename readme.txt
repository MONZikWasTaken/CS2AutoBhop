туториал на колесико:
кидает 3 файла "+jump.cfg", "-jump.cfg", "autoexec.cfg" 
в путь к Counter-Strike Global Offensive\game\csgo\cfg (прям сюда), 
перезапускаем игру, запускаем autofps.ahk (ahk V1.1 должен быть установлен),
и вот, у вас идеальный бхоп на колесико

туториал на автобхоп:
кидает 3 файла "+jump.cfg", "-jump.cfg", "autoexec.cfg" 
в путь к Counter-Strike Global Offensive\game\csgo\cfg (прям сюда), 
перезапускаем игру, запускаем autobhop.ahk (ahk V1.1 должен быть установлен),
в config.ini указываем свою кнопку для автобхопа (изначально стоит space/пробел)
теперь при нажатии на space у вас будет спамится колесико мыши и бхоп будет работать.

AutoBhopSpace = автобхоп без колесика, чисто на пробел, хуево работает но хоть как то хд
